## 概要
- `TypeScript 6.0.3 に更新` をリバートします。
- main マージ後の GitHub Actions が `safe-chain` の minimum package age により `typescript@6.0.3` を取得できず失敗したため、復旧を優先しました。

## 失敗内容
- 失敗した Actions: <https://github.com/kurehajime/inyago/actions/runs/24594497194>
- `npm ci` 時に `typescript@6.0.3` が safe-chain にブロックされ、GitHub Pages の deploy workflow が失敗しました。

## 検証
- `npm test`
  - `test` スクリプト未定義のため、テストコードなしとして扱いました。
- `npm run build`
  - 成功
- `npm run lint`
  - 成功
